GIT-kommandoer


git add [filnavn] - Gjør filen du har valgt klar for å bli commited. 

git commit -m "[message]" - Commiter filene du har lagt til med 'add' til din valgte branch

git push origin [branch-navn] - Pusher din lokale branch opp til github

git status - Kommando for å sjekke hvilke filer du har 'addet', men ikke commited enda

git diff - For å sjekke forskjellene på din lokale branch og master-brancen.

git branch - Sjekke hvilke brancher du har og hvilken du har valgt

git branch [branchnavn] - Lager ny branch

git branch -d [branchnavn] - Sletter branchen

git branch -m [branchnavn] - Endrer navnet på branchen

git checkout [branchnavn] - Bytter til den branch [branchnavn]

git checkout -b [branchnavn] - Lager ny branch OG bytter til den nye branchen i en kommando

git reset HEAD - Dersom du har 'addet' filer for å bli commited, men angrer så kan du bruke denne kommandoen for å fjerne de 'addete' filene.
